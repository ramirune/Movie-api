Documentation Page

To build the server-side component of a “movies” web application. The web application will provide users with access to information about different movies, directors, and genres. Users will be able to sign up, update their personal information, and create a list of their favorite movies.

Request URL Responses
GET /movies Get a list of ALL movies.
GET /genres Get a list of all genres.
GET /genres/:Name Get a genre information by genre's name
GET /movies/:Title Get a movie's data by title.
GET /movies/Genre/:Name Get movies' data by genre.
GET /directors Get a list of all directors
GET /directors/:Name Get a director's informations by name.
POST /users Add a new user.
GET /users Get a list of all users.
GET /users/:Username Get a user's informations by username.
PUT /users/:Username Allow user to update the user information
POST /users/:Username/movies/:MovieID Allow user to add a movie to the list of favorites.
DELETE /users/:Username/movies/:MovieID All user to remove a movie from the list of favorites.
DELETE /users/:Username Allow existing user to deregister.
